a small presentation about the
Battleship game project

Battleship is a classic text-based game written in the C programming language, where two players compete in a strategic naval battle. The game features a virtual (X ; Y) board where ships are placed. The objective of the player is to sink all of the opponent's ships by correctly guessing their locations

One of the peculiarities of my game is the free change of the size of the playing field.
So it can be 5x5 7x7 10x10

This is how the game board looks like

You can move and place ships using the keyboard, namely the left-right, up-down, and enter buttons, and you can also attack

This is how the game board looks now when ships are placed

The computer itself places ships that we do not see,
and it can also attack

Blue is the sea.
Gray is "miss".
Yellow is the cursor.
And green is "hit".
